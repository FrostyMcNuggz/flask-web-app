About This App

This web application was authored in VS Code. It was built using Python, Flask, and the Bootstrap 5.3 Framework. It utilizes an SQLite3 database, and runs in a virtual environment. Version control is implemented via Github. Containerized deployment on a Virtual Private Server via Docker Hub is the end-goal.

This web application is extrapelated from, and expounds upon the Python Tutorial found here.
